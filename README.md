# gestion-pharmacie-java
 Système de Gestion de Pharmacie : Application de bureau (Desktop) développée en Java (Swing) avec base de données MySQL, suivant l'architecture MVC. Gère le stock de médicaments (alertes de péremption et de stock faible), les ventes, la facturation automatique (.txt), les clients, les ordonnances et les commandes fournisseurs.
